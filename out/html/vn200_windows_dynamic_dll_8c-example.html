vn200_windows_dynamic_dll.c

This example is a conversion of the vn200_windows_basic.c so that it uses a dynamically loaded DLL provided by the library. The Windows API is used via the windows.h header file to load the DLL into memory using the function LoadLibrary and then pointers to the functions provided by the DLL are retrieved individually using the function GetProcAddress.
